# 欧派AI灵感家

基于成交案例库的客户需求洞察与全案推荐系统，用于“AI + 客户体验”比赛MVP演示。

系统通过七轮视觉选择帮助客户表达模糊需求，实时形成客户画像，识别需求冲突，推荐三套可解释方案，并将所选方案带入八阶段交付看板和风险预警流程。

## 已实现能力

- 七轮视觉偏好选择：喜欢、不喜欢、都可以、跳过、撤销上一步
- 实时客户画像与“20%开放＋80%封闭”冲突建议
- 基于60条脱敏模拟案例的三策略推荐
- 七项固定权重评分、推荐理由和预算重算
- 量尺至验收八阶段模拟交付看板
- 报价漏项与跨品类交期不一致风险预警
- 30张明确标注为“AI生成示意图”的演示图片
- Express、Zod与Prisma接口和数据模型边界

## 本地运行

要求 Node.js `>=22.13.0`。

```powershell
Set-Location site
npm.cmd install
npm.cmd run dev
```

启动可选Express API：

```powershell
Copy-Item .env.example .env
npm.cmd run api:dev
```

更完整的功能、数据边界、演示流程和故障排查请查看 `outputs/欧派AI灵感家_系统使用说明书_v1.0.docx`。

## 数据声明

本仓库用于比赛演示。客户、预算、成交、满意度、SKU、价格、日期、进度和风险事件均为脱敏模拟或AI虚构数据，不连接欧派真实客户、订单、报价、生产或交付系统。
